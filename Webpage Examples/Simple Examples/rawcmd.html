Send raw binary commands

This page can be used to send binary serial commands to the drive.

The details of the binary serial command format can be found in the binary serial command applications note on Copley's web site. Every command has an 8-bit op-code and takes zero or more 16-bit words of data.

In the table below, enter the opcode and data (comma seperated) to send to the drive and the drive's response will be displayed.

For example, to get parameter 0 (current loop KP), enter 12 in the opcode field (the code for 'get') and 0 in the data field.

To set the current loop kp to 123, enter 13 in the opcode field and 0,123 in the data field.
