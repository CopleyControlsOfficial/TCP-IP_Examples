Get / Set Parameters

This page allows you to read and write drive parameters.

Enter one or more parameter ID values in the first column and press the 'Get parameter' button to read the current value of the parameters.

Update the value in the second column and press the 'Set parameter' button to update the parameter values.

The parameter dictionary (available on Copley's web site) lists the meaning of the various drive parameters.
